# Parkinson-s-Disease-Detection
Parkinson's disease is a nervous system disorder that impairs movement control.The disease typically begins slowly and progresses over time. Parkinson's disease affects about half as many men as it does women.


Changes in a person's speech are one of the most common symptoms of Parkinson's disease.Speech evolves. OneÂ  can speak softly, quickly, slur, or pause before speaking. Ones speech may be monotone rather than with the typical inflections.The dataset contains records of biomedical voice measurements.This project's goal is to use biomedical voice measurements to determine whether a person has Parkinson's disease.
